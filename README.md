# 🧮 Calculadora de Apuestas de Sistema + Value Betting

**Herramienta profesional para apuestas deportivas (fútbol)**  
Calcula la mejor distribución de sistemas (Trixie, Yankee, Dobles + Triples, etc.) y te muestra el **Value Betting** real según tu probabilidad estimada.

Inspirado en la calculadora de surebets de [casasdeapuestas.bet](https://casasdeapuestas.bet/cl/calculadora/), pero **adaptado 100% a apuestas de sistema** para minimizar riesgo y maximizar ganancia a largo plazo.

### ✨ Características

- **Value Betting integrado**: Ingresa tu probabilidad estimada real y ve el Edge % y EV en cada selección.
- **Sistemas automáticos**: Dobles, Triples, Trixie, Patent, Yankee, Lucky 15, Dobles+Triples y Parlay.
- **Recomendación inteligente**: Elige automáticamente el sistema con **mayor EV** y **menor riesgo** (break-even).
- **Distribución exacta** del stake (copia y pega directamente en Betano).
- **Escenarios de aciertos**: Ganancia mínima/máxima según cuántos aciertes.
- **Interfaz moderna y responsive** (funciona perfecto en móvil).
- Sin instalación: solo abre el enlace.

### Cómo usar (en menos de 30 segundos)

1. Abre la [app web](https://calculadora-sistemag.streamlit.app).
2. Elige el número de selecciones (3 a 8).
3. Ingresa las **cuotas** de cada partido.
4. Ingresa tu **probabilidad estimada real** (%) para cada una.
5. Pon el monto total a invertir.
6. Pulsa **“🚀 Calcular”**.
7. Ve la pestaña **“✅ Recomendado”** → copia la distribución y pégala en Betano.
